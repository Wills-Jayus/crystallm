====================
三、Step 2：优化 constrained decoding
====================

新增一个改进版采样器：

scripts/sample_symcif_v2_constrained_v2.py

不要推翻原采样器，保留旧版作为 old。

重点优化以下部分：

1. Wyckoff skeleton 多样性

对同一个 test sample 的 n=5 生成：
- 记录已生成的 skeleton signature；
- skeleton signature 至少包含：
  (element, wyckoff_letter, multiplicity) sorted or canonicalized
- 如果新生成 skeleton 与已有 skeleton 完全重复，允许有限次数 resample；
- 避免 5 个 candidate 全部是同一个 skeleton。

目标：
提高 match@5，而不是只提高单条合法率。

2. Wyckoff letter 选择改进

在生成 Wyckoff letter 时，继续满足：
- letter 属于 target SG；
- multiplicity <= remaining_counts[element]；
- 选择后 remaining_counts 仍可闭合；
- fixed WP 不非法重复。

新增：
- 优先按照 CrystalFormer 思想，从高 site symmetry / 小 multiplicity / 字母靠前的 special positions 开始；
- 但不要完全贪心，要保留采样多样性；
- 可以对合法候选按模型概率重新归一化采样；
- 如果候选过少，再放宽 diversity resample，而不是破坏 formula closure。

3. Element-Wyckoff assignment 改进

生成 element 时：
- 只允许 remaining_counts > 0 的元素；
- 对剩余 count 较难闭合的元素优先生成；
- 可选策略：优先处理 remaining count 小、可选 multiplicity 少的元素；
- 不要让模型长期拖到最后才生成难闭合元素。

4. Free coords / cell 低温采样

实现 field-specific sampling config：

默认：
- element / Wyckoff letter: temperature=1.0, top_k=10
- free coordinates: temperature=0.7, top_k=5
- cell parameters: temperature=0.5 or 0.7, top_k=5

目标：
降低 RMSE，同时尽量不牺牲 match@5。

请支持命令行参数：
--temp-discrete
--topk-discrete
--temp-coord
--topk-coord
--temp-cell
--topk-cell

5. Cell projection 加强

生成 cell 后按晶系做 deterministic projection：
- cubic: a=b=c, alpha=beta=gamma=90
- tetragonal: a=b, alpha=beta=gamma=90
- orthorhombic: alpha=beta=gamma=90
- hexagonal/trigonal hex setting: a=b, alpha=beta=90, gamma=120
- monoclinic/triclinic 先保守处理

volume 一律根据 projected cell 重新计算。

6. 输出日志

每条 generation 记录：
- raw generated SymCIF-v2
- constrained state trace
- remaining_counts trace
- selected Wyckoff sites
- skeleton signature
- number of resample attempts
- mask rejection reasons
- final standard CIF path
