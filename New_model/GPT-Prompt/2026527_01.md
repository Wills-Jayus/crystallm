========================================
Goal 1：重构动作空间，禁止 58,880 类直接 action softmax
========================================

目标：
实现分层结构化 decoder。

不要再使用：
element@OrbitToken 直接作为 58,880 类 action。

改成分层输出：
1. orbit_head:
   预测 OrbitToken / Wyckoff row token / stop。

2. element_head:
   在当前 remaining formula 中选择元素。
   element 只能来自 formula 中尚未用完的元素。

3. multiplicity / formula closure constraint:
   每步选择 OrbitToken 后，根据 multiplicity 扣减 remaining composition。
   不允许 remaining composition 为负。
   不允许选择会导致后续无法 exact-cover 的 OrbitToken。
   stop 只能在 formula 完全闭合时允许。

4. free_param_head:
   只预测该 OrbitToken 的 active free_param dimensions。
   对 inactive dimensions 不计算 loss。

5. lattice_head:
   预测 normalized lattice。
   使用 crystal-system-aware normalization/mask。

decoder state 必须包含：
- formula embedding
- SG embedding
- remaining composition vector
- step index
- chosen orbit multiset summary
- legal OrbitToken mask
- exact-cover feasibility features, if easy
- previous row embedding

输出：
reports/symcif_v4_mp20_minicfjoint_v2/01_action_space_design.md
