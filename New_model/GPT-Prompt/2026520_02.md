====================
五、实现 CrystalFormer-style constrained decoding
====================

这是本阶段最核心任务。

新增脚本：

scripts/sample_symcif_v2_constrained.py

采样时维护状态：

target_formula_counts
remaining_counts
target_sg
valid_wyckoff_letters
used_fixed_wyckoff_positions
current_site_element
current_site_letter
current_section

生成逻辑：

1. 从 prompt 解析目标 formula 和 target SG。
2. 初始化 remaining_counts = target_formula_counts。
3. 进入 Wyckoff site loop。
4. 生成 element 时，只允许：
   remaining_counts[element] > 0

5. 生成 Wyckoff letter 时，只允许：
   - letter 属于当前 target SG；
   - multiplicity(SG, letter) <= remaining_counts[current_element]；
   - 如果该 Wyckoff position 是不可重复 fixed site，不能重复使用；
   - 选择该 letter 后，remaining_counts[current_element] - multiplicity 之后，剩余计数仍然有机会由当前 SG 的 Wyckoff multiplicity 组合闭合。

6. formula closure 检查：
   实现一个简单 DP / backtracking 函数：
   can_close(remaining_counts, sg_number, available_wyckoff_multiplicities)
   如果某个候选会导致未来无法闭合公式，mask 掉。

7. 生成一个 site 后：
   remaining_counts[element] -= multiplicity(SG, letter)

8. derived fields 不让模型生成：
   - multiplicity
   - site_symmetry
   - enumeration
   - fixed/free mask
   都由 lookup 自动决定。

9. 坐标生成：
   - 固定坐标：直接写 FIXED，不采样。
   - 自由坐标：允许模型生成数值。
   - 自由坐标必须为合法浮点数，归一化到 [0,1)，保留四位小数。
   - 如果生成失败，可以重采当前坐标，但不要用 GT。

10. STOP / 进入 cell section：
    只有 remaining_counts 全部为 0 时，才允许输出 <END_SITES> 或进入 _cell_length_a。
    如果 remaining_counts 未闭合，禁止结束 Wyckoff loop。

11. cell 参数生成：
    可以先使用普通采样。
    但根据 SG 所属晶系做投影：
    - cubic: a=b=c, alpha=beta=gamma=90
    - tetragonal: a=b, alpha=beta=gamma=90
    - orthorhombic: alpha=beta=gamma=90
    - hexagonal/trigonal hex setting: a=b, alpha=beta=90, gamma=120
    - 其它晶系先保守处理。

12. 采样完成后转标准 CIF，并走原 evaluator。
